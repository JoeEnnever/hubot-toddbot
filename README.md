# hubot-toddbot

A replacement for Todd

See [`src/toddbot.coffee`](src/toddbot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-toddbot --save`

Then add **hubot-toddbot** to your `external-scripts.json`:

```json
[
  "hubot-toddbot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
